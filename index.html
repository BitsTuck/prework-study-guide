HTML

  • The Head element contains all of the metadata for the page
  • The only thing that displays from the Header section is the page title
  • The H1 tag should only be used once per page
  • All elements should be labelled so that they can be better indexed on the internet.

CSS

  • There are three ways to apply style sheets to a page
  • The best practice is to have an external style sheet applied in the header of each page
  • Margin is the space outside of an element
  • Padding is the space within an element

Git

  • Git Status checks current branch
  • Git Checkout moves you to a new branch
  • Git Pull brings in the most up-to-date code
  • Git Push sends updates to the repo
